# 💫 About Me:
I'm a young French developer, still in high school. I code very regularly; it's a pastime, a hobby, and a passion. I aspire to make it my career later on. I learn a lot on my own, by experimenting, trying things out, and learning from my mistakes.<br><br>When I code, I sometimes use AI, but I believe it's really important to know what you're writing instead of just blindly copying. That's why all the code on this GitHub repository is code I understand and can explain. I've always been curious, and understanding code with the help of AI really helps me. I feel lucky to live and grow up in a time when we have these kinds of tools.<br><br>Regarding my skills: I've always used Linux daily; I've literally never used any other operating system. Today I also use Git and GitHub, a large majority of my projects are open source, I have experience using Python and HTML/CSS, I also use SQLite for databases, I sometimes use JSON and YAML, and right now I'm trying to learn (still self-taught) Go
